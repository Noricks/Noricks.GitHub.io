GeoJSON on Python

Noricks | 周一 17 二月 2020

为了排雷和学习，在此记录一下Python地理信息库GEOJSON的安装使用的学习过程

安装

python 环境

conda create -n geojson python=3.7.0

Geojson

pip install geojson
Github: https://github.com/jazzband/geojson

扩展包 GeoPandas

坑1

使用pip install geopandas安装 会出现
A GDAL API version must be specified. Provide a path to gdal-config using a GDAL_CONFIG environment variable or use a GDAL_VERSION environment variable. 错误提示 错误来源应该是fiona这个前置包
解决方法: 可以参考 https://blog.csdn.net/lionhenryzxxy/article/details/101423206

### 坑2 为了解决附属包安装错误的问题尝试使用 conda install fiona
注意: conda 并没有 geopandas 的检索 而问题的确出在 fiona 的安装上 再使用pip安装就可以解决问题了
（包都能在清华源上自动被pip找到） ### 坑3 在conda安装过程中出现conda无法移除部分文件夹需要手动释放移除的(need reboot)问题
重启并删除了环境，重新安装就解决了
